# Firefox Setup

## Navigation
`ctrl + i` -> page info (perms, encryption, etc)
`ctrl + k` -> search bar
`ctrl + tab` -> cycle through tabs

## Extensions
- uBlock Origin

## Settings
- vertical tabs
    - `Settings > General > Browser Layout > Check Vertical Tabs`
- remove recommendations
    - `Settings > General > Browsing > Uncheck "Recommend {extensions, features} as you browse"`

- keep new tab page as blank
